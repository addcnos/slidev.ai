# 快速开始 {#getting-started}

## 安装 {#installation}

Slidev AI 是一个基于 Electron 的桌面应用程序。您可以通过以下步骤来安装和运行它：

1. 克隆 Slidev AI 仓库:
   ```sh
   git clone https://github.com/your-repo/slidev-ai.git
   cd slidev-ai
   ```

2. 安装依赖:
   ```sh
   yarn
   ```

## 开发 {#development}

要在开发模式下运行 Slidev AI，请执行以下命令：
```
yarn dev
```

## 构建 {#build}

要构建 Slidev AI 的可执行文件，请执行以下命令：
```sh
yarn build
```
